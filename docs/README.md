# Documentation

This directory contains all project documentation including the Software Requirements Specification (SRS), system diagrams, and presentation materials.

## Contents

| File | Description |
|---|---|
| `SRS.docx` | Software Requirements Specification — functional and non-functional requirements of the system. |
| `ERD.png` | Entity-Relationship Diagram — visual representation of the database schema. |
| `System_Workflow.png` | High-level system workflow diagram showing how all components interact. |
| `Sequence_Diagram.png` | Sequence diagram illustrating the key interaction flows between system actors. |
| `Presentation.pptx` | Project presentation slides used for the graduation defence. |
